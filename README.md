- Support VPS & Termux 
- Proky Support
- Multi account Support
- Bot Auto Daily to get exp
  
*🖥️ Start installation*

````
git clone https://github.com/ZonaAirdrop/Rinso-Bot-daily.git
cd Rinso-Bot-daily
npm install
````

````
nano proxies.txt
````

````

````
